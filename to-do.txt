======
Пересказчик видео
======
Versions:
python 3.11.9

libs:
openai-whisper
moviepy == 1.0.3
openai == 1.30.2
======
Для запуска через консоль:
1. Установить Python 3.11.9
2. В дериктории запустить cmd:> pip install -r requirements.txt
3. Запустить программу cmd:> python3 main.py
======
Работа в программе:

Программа представляет из себя преобразование аудио/видео формата в формат текста. Преобразованный текст далее с помошью промпта отправляется ChatGPT, который в своб очередь дает краткий пересказ текста.
Для работы необходим GptApiKey вида sk-proj-********
